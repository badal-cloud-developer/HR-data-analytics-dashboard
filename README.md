# HR Data Analysis Dashboard

A Power BI dashboard for analyzing workforce data, employee attrition trends, and job satisfaction across departments and age groups.

---

## Overview

This dashboard provides HR teams and management with a clear view of employee demographics, attrition patterns, and satisfaction metrics — enabling data-driven decisions around retention and workforce planning.

---

## Key Metrics

| Metric | Description |
|---|---|
| **Overall Employees** | Total headcount in the organization |
| **Active Employees** | Currently employed workforce |
| **Attrition** | Number of employees who left |
| **Attrition Rate** | Percentage of employee turnover |
| **Average Age** | Mean age across the workforce |

---

## Dashboard Visuals

- **Department-Wise Attrition** — Pie chart breaking down attrition across departments
- **Employees by Age Group** — Column chart showing headcount distribution across age bands
- **Job Satisfaction Rating** — Pivot table mapping satisfaction scores by role
- **Attrition by Age Group** — Donut charts segmented by age bands (Under 25, 25–34, 35–44, 45–54, Above 55)
- **Interactive Slicer** — Filter all visuals dynamically by department or other dimensions

---

## Tools & Technologies

- **Power BI Desktop** — Dashboard design and publishing
- **DAX** — Calculated measures and KPIs
- **Data Source** — HR employee dataset (CSV/Excel)

## How to Use

1. Open `HR_data_analysis_dashboard.pbix` in Power BI Desktop.
2. If prompted, refresh the data source connection.
3. Use the slicer panel to filter by department or other dimensions.
4. Hover over charts for detailed tooltips and drill-through options.
